## 🔧 使用方式

### 1. 环境准备
```bash
# 安装依赖
pip install -r requirements.txt

# 安装ChromeDriver（电商爬虫需要）
# 下载地址：https://chromedriver.chromium.org/
```

### 2. 快速开始
```bash
# 查看可用爬虫
python main.py --list

# 运行新闻爬虫
python main.py --run news

# 运行电商爬虫演示
python main.py --demo-ecommerce

# 运行所有爬虫（演示模式）
python main.py --demo-all
```

### 3. Python代码示例
```python
from main import CrawlerManager

# 创建爬虫管理器
manager = CrawlerManager()

# 查看可用爬虫
crawlers = manager.list_crawlers()
print(f"可用爬虫: {len(crawlers)}个")

# 运行特定爬虫
result = manager.run_crawler("news", sources=["sina", "tencent"], max_articles=10)
print(f"爬取到 {len(result['data'])} 条新闻")

# 导出数据
manager.export_data(result, "news_data.xlsx")
```
