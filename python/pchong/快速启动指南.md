## 快速开始

### 1. 环境准备

```bash
# 克隆项目（如果尚未克隆）
git clone <repository-url>
cd pchong

# 创建虚拟环境（推荐）
python -m venv venv

# 激活虚拟环境
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

# 安装依赖包
pip install -r requirements.txt
```

### 2. 运行测试

```bash
# 运行完整测试套件
python test_all_crawlers.py

# 或单独测试新闻爬虫
python -m crawlers.news.news_crawler
```

### 3. 使用新闻爬虫

```python
# 方法1: 使用命令行
python main.py --run news

# 方法2: 在Python代码中使用
from crawlers.news.news_crawler import NewsCrawler

crawler = NewsCrawler()
data = crawler.run(
    sources=["sina", "tencent"],  # 新闻源
    categories=["technology"],     # 分类
    max_articles=10                # 最大文章数
)

print(f"爬取了 {len(data)} 条新闻")
```

### 4. 使用电商爬虫

```python
# 方法1: 使用命令行（演示模式）
python main.py --demo-ecommerce

# 方法2: 在Python代码中使用
from crawlers.ecommerce.ecommerce_crawler import EcommerceCrawler

crawler = EcommerceCrawler()
data = crawler.run(
    platforms=["taobao", "jd"],  # 电商平台
    category="electronics",       # 商品分类
    max_pages=2,                  # 最大爬取页数
    products_per_page=20          # 每页商品数
)

print(f"爬取了 {len(data)} 个商品")
```

**注意**: 电商爬虫需要Chrome浏览器和ChromeDriver，实际爬取可能需要VPN。

## 核心模块说明

### 1. 基础爬虫类 (`crawlers/base/base_crawler.py`)

- 所有爬虫的基类
- 提供HTTP请求、重试机制、日志记录等通用功能
- 支持随机User-Agent和请求延迟

### 2. 新闻爬虫 (`crawlers/news/news_crawler.py`)

- 支持新浪、腾讯、网易、今日头条四大新闻源
- 可爬取热点新闻和分类新闻（科技、财经、体育、政治）
- 自动解析新闻标题、内容、发布时间、作者等信息

### 3. Excel导出器 (`utils/excel_exporter.py`)

- 将数据导出为格式化的Excel文件
- 自动设置列宽、样式、条件格式
- 生成统计信息和工作表

### 4. 主控制程序 (`main.py`)

- 统一的爬虫管理和运行界面
- 支持命令行参数
- 可单独运行或批量运行爬虫

## 输出文件说明

运行爬虫后，会在 `output/` 目录生成：
1. `news_YYYYMMDD_HHMMSS.xlsx` - Excel数据文件
2. `crawler_report_YYYYMMDD_HHMMSS.json` - 运行报告

Excel文件包含：
- **爬取数据**工作表：详细的爬取数据
- **爬虫统计**工作表：运行统计信息
- **数据概览**工作表：数据摘要和样本

## 注意事项

1. **遵守法律法规**: 爬虫开发需遵守目标网站的robots.txt和相关法律法规
2. **控制请求频率**: 避免对目标网站造成过大压力
3. **数据使用**: 合理使用爬取的数据，避免侵犯隐私和版权
4. **反爬虫策略**: 不同网站有不同的反爬虫机制，需要相应的应对措施
