## 🚀 使用方式

### 基本使用
```python
from crawlers.travel.travel_crawler import TravelCrawler

# 创建爬虫
crawler = TravelCrawler()

# 搜索酒店
result = crawler.run(city="北京", platform="all", max_hotels=50)

# 导出数据
filepath = crawler.export_data(result)
```

### 命令行使用
```bash
# 查看所有爬虫
python main.py --list

# 运行旅游爬虫
python main.py --run travel --city 上海 --check_in 2024-12-01 --check_out 2024-12-07

# 运行演示
python main.py --demo
```

## 🔧 配置系统

### 配置文件 (`config/settings.py`)
```python
TRAVEL = {
    "max_hotels": 50,          # 最大爬取酒店数
    "check_in": "2024-12-01",  # 默认入住日期
    "check_out": "2024-12-07", # 默认离店日期
    "locations": ["北京", "上海", "广州", "杭州"]  # 默认城市列表
}
```

### 运行参数
```python
crawler.run(
    city="上海",                # 搜索城市
    check_in="2024-12-01",     # 入住日期
    check_out="2024-12-07",    # 离店日期
    platform="all",            # 平台：ctrip/qunar/fliggy/all

)
```
